INSTALLATION INSTRUCTIONS

To install for the local user, open the Arduino.mode file in Coda 2, and it will ask if you want to install the mode. It gets installed to: ~/Library/Application Support/Coda 2/Modes

Open the Arduino.seemode (for SubEthaEdit 4 and above) and it will open the file in SEE and ask to install it. It gets placed in: ~/Library/Containers/de.codingmonkeys.SubEthaEdit.Mac/Data/Library/Application Support/de.codingmonkeys.SubEthaEdit.Mac/Modes

The following may work for all users: To Install for all users, the mode file, Arduino.mode, needs to be placed in: for Coda: /Library/Application Support/Coda/Modes/ for SubEthaEdit: /Library/Application Support/SubEthaEdit/Modes/

Quit the application, if it is running, and open it up again.
